This zip contains the sample code for the Pluralsight Course,
C# Equality and Comparisons 
by Simon Robinson
Module 9 - Hash Codes and Hashtables

All code has been created in VS2010 and tested in VS2010 and VS2013.

ValueTypeEquality: Copy of the project of the same name from module 4 - included here to illustrate a implementation of GetHashCode()
FoodNameEquality: Demonstrates an equality comparer that does not check all fields in a type
EqualityComparerDemo: Extension of the project of the same name from module 8 - extended to show using MS StringComparer to supply hash codes
