URL Rewrite for IIS will run on IIS 7.0 and later, which includes Windows Vista, Windows 7, Windows 8, Windows 8.5 and later, and Windows Server 2008, Windows Server 2008R2, Windows Server 2012, and Windows Server 2012 R2.

It will also work with most web hosting companies who support IIS 7.0 or greater. If needed, contact your hosting company to ensure that they support URL Rewrite.
