The HelloWorldODP.xml file is the XML extracted from an OpenDocument Format
file of a one-slide presentation.  Note the declaration of 38 namespace
prefixes in the document element.

The UBL Invoice example from the slides is found at:
  http://docs.oasis-open.org/ubl/os-UBL-2.1/xml/UBL-Invoice-2.1-Example.xml
    
The genericode example from the slides is found at:
  http://docs.oasis-open.org/ubl/os-UBL-2.1/cl/gc/default/CurrencyCode-2.1.gc
  
The context/value association example from the slides is found at:
  http://docs.oasis-open.org/ubl/os-UBL-2.1/cva/UBL-DefaultDTQ-2.1.cva
