This zip contains the sample code for the Pluralsight Course,
C# Equality and Comparisons 
by Simon Robinson
Module 8 - Comparers and Equality Comparers

All code has been created in VS2010 and tested in VS2010 and VS2013.

SortItems: Demonstrates using a comparer to change how Food instances are sorted
InconsistentSorting: Demonstrates how sort order can be inconsistent if you don't check all fields when comparing
EqualityComparerDemo: Demonstrates implementing an equality comparer for the FoodItem struct
StringComparers: Demonstrates using an MS string comparer to change how a collection sorts strings
