This zip contains the sample code for the Pluralsight Course,
C# Equality and Comparisons 
by Simon Robinson
Module 7 - Strings

All code has been created in VS2010 and tested in VS2010 and VS2013.

OrdinalCompareStrings: Demonstrates string ordinal comparisons
CompareStringsDemo: Demonstrates how different types of comparison change the results when comparing strings
StringEquality: Demonstrates different ways to check strings for equality
